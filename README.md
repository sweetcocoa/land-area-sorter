# Land Area Sorter

지원되는 부동산 사이트의 검색 결과를 전용면적 기준으로 다시 정렬해 보는 개인용 크롬 확장입니다.

## 설치

1. Chrome에서 `chrome://extensions`를 엽니다.
2. 우측 상단 `개발자 모드`를 켭니다.
3. `압축해제된 확장 프로그램을 로드합니다`를 누릅니다.
4. 이 프로젝트 폴더를 선택합니다.

## 사용 방법

1. `https://new.land.naver.com/`에서 매물 목록이 보이는 페이지를 엽니다.
2. 정렬 영역 근처에 `전용면적순` 버튼이 나타납니다.
3. 버튼을 누를 때마다 다음 순서로 바뀝니다.
   - 원본순
   - 전용면적 오름차순
   - 전용면적 내림차순

정렬 UI를 찾지 못한 경우에는 화면 우측 상단에 플로팅 버튼으로 표시됩니다.

## 동작 방식

- 매물 카드의 텍스트에서 `전용` 라벨을 우선 파싱합니다.
- `공급/전용` 형태만 있으면 두 번째 값을 전용면적으로 사용합니다.
- 파싱에 실패한 카드는 원래 위치를 유지합니다.
- 무한 스크롤, 필터 변경, 지도 이동 등으로 목록이 다시 그려져도 현재 정렬 상태를 다시 적용합니다.

## 한계

- 네이버 부동산 DOM 구조가 크게 바뀌면 선택자 보완이 필요할 수 있습니다.
- 현재 버전은 Chrome 저장소를 사용하지 않아서 새로고침 후에는 `원본순`부터 시작합니다.
